# Proyecto 3 – Análisis del desempeño financiero con SQL

## 🎯 Objetivo
Evaluar el desempeño financiero de la empresa mediante consultas SQL que permitan identificar ingresos, gastos y utilidades clave.

- Fuente: [indica si fue proporcionado por el bootcamp, Kaggle, etc.]
- Tamaño: [ejemplo: 100,000 registros]
- Variables principales: fecha, categoría de gasto, ingresos, región, departamento.

## 🔧 Herramientas utilizadas
- SQL (PostgreSQL / MySQL)
- Consultas de agregación y funciones de ventana
- Exportación de resultados a Excel / Power BI

## 📊 Consultas principales
1. Ingresos totales por trimestre y año.
2. Gastos operativos por departamento.
3. Margen de utilidad neta.
4. Comparación de desempeño entre regiones.

## 📈 Resultados
- Identificación de las áreas con mayor rentabilidad.
- Detección de gastos que afectan la utilidad neta.
- Resumen ejecutivo para la toma de decisiones financieras.
